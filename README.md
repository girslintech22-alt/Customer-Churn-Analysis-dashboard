
# Customer Churn Dashboard – Data Analysis Report

## Project Overview

This project analyzes customer churn using an interactive Power BI dashboard. The goal is to understand **why customers leave (churn)** and identify patterns that can help improve customer retention.

The dashboard provides insights into customer behavior based on:

* Contract type
* Payment method
* Internet services
* Tenure and charges

All insights below are derived from the dashboard visuals provided in the report .

---

## Key Metrics Summary

| Metric          | Value     |
| --------------- | --------- |
| Total Customers | 7,000     |
| Churn Customers | 2,000     |
| Churn Rate      | 26.54%    |
| Total Revenue   | ~$456K    |
| Average Tenure  | 32 months |

### What this means

* Around **1 in 4 customers are leaving**, which is a high churn rate.
* The business is generating good revenue, but **customer retention needs improvement**.

---

## Customer Demographics

* Gender distribution is almost equal:

  * Male: 3.49K
  * Female: 3.56K

### Insight

* **Gender does not significantly impact churn**, so other factors are more important.

---

## Churn by Tenure

| Tenure    | Churn Count |
| --------- | ----------- |
| < 1 Year  | 1037        |
| < 2 Years | 294         |
| < 4 Years | 325         |
| 5+ Years  | 213         |

### Insight

* Most churn happens in the **first year**.
* Customers who stay longer are **less likely to leave**.

Conclusion:
**Early customer experience is critical.**

---

## 💳 Payment Method Analysis

| Payment Method   | % of Customers |
| ---------------- | -------------- |
| Electronic Check | 45.29%         |
| Mailed Check     | 19.11%         |
| Bank Transfer    | 16.71%         |
| Credit Card      | 15.24%         |

### Insight

* Customers using **electronic check show higher churn behavior**.
* More stable methods (like credit card, bank transfer) show **better retention**.

Conclusion:
Encouraging **automatic payment methods** may reduce churn.

---

## Contract Type Analysis

| Contract Type  | %      |
| -------------- | ------ |
| Month-to-Month | 42.71% |
| One Year       | 11.27% |
| Two Year       | 2.83%  |

### Insight

* **Month-to-month contracts have the highest churn**.
* Long-term contracts (1–2 years) have **lower churn**.

 Conclusion:
Customers without commitment are **more likely to leave**.

---

## Internet Service Analysis

From the dashboard (page 2):

* DSL churn rate: **18.96%**
* Fiber and other services also contribute significantly

### Additional Metrics:

* DSL Monthly Charges: ₹140.67K
* Total Charges: 5.12M

* Internet service type impacts churn.

## 📺 Service Usage Insights

| Service          | Usage % |
| ---------------- | ------- |
| Phone Service    | 90.32%  |
| Streaming Movies | 38.79%  |
| Streaming TV     | 38.44%  |
| Tech Support     | 29.02%  |

### Insight

* Customers without **tech support** are more likely to churn.
* Add-on services like streaming have moderate adoption.

 Conclusion:
Offering **better support services** can reduce churn.

---

## Churn vs Charges & Contract

From the combined chart (page 2):

* **Month-to-month customers** show:

  * High churn rate
  * Lower long-term value

* **Two-year contracts**:

  * Low churn
  * Higher customer lifetime value (CLV)

### Insight

* Longer contracts = **higher stability and revenue**
* Short-term customers = **higher risk**

---

## Customer Lifetime Value (CLV)

* Highest CLV: **Two-year contracts**
* Lowest CLV: **Month-to-month customers**

### Insight

* Long-term customers are **more profitable**
* Retaining customers increases revenue over time

---

## Key Business Insights

1. **High churn rate (26.54%) is a major concern**
2. Most churn happens in the **first year**
3. **Month-to-month contracts drive churn**
4. **Electronic check users churn more**
5. Lack of **tech support increases churn**
6. Long-term customers bring **higher value**

---

## Recommendations

* Improve **onboarding experience** for new customers
* Offer **discounts for long-term contracts**
* Promote **automatic payment methods**
* Provide **better customer support services**
* Target high-risk customers with **retention campaigns**
